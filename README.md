# 📊 Data Professional Survey Analysis

## Project Overview
Analysis of a global survey of 630 data professionals to understand 
salary distributions, job satisfaction, favourite tools, and how 
difficult it is to break into the data field across different roles.

## Tools Used
- **Microsoft Excel** — data cleaning and preparation
- **Power BI** — interactive dashboard and visualisation

## Key KPIs
| Metric | Value |
|---|---|
| Total Survey Takers | 630 |
| Average Age | 29.87 years |
| Happiness with Work/Life Balance | 5.74 / 10 |
| Happiness with Salary | 4.27 / 10 |

## Key Insights
- **Data Scientists** earn the highest average salary across all roles
- **Python** is the most favourite programming language by a large margin
- **42.7%** of respondents found breaking into data neither easy nor difficult
- **31.74%** found it difficult or very difficult to break in
- Only **25.56%** found it easy or very easy
- Salary satisfaction scores **4.27/10** — majority feel underpaid
- Work/life balance scores **5.74/10** — moderate satisfaction
- Majority of survey takers are based in the **United States**
- Average age of data professionals surveyed is **29.87 years**
